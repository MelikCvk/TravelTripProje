## 📌 Proje Hakkında

**TravelTrip**, ASP.NET MVC5 mimarisi ile Code First Entity Framework kullanılarak geliştirilmiş bir seyahat blog web uygulamasıdır. Kullanıcılar blog yazılarını okuyabilir, yorum yapabilir ve iletişim formu aracılığıyla site sahibiyle iletişime geçebilir. Admin paneli üzerinden tüm içerikler yönetilebilir.

## 🚀 Özellikler

### 👤 Kullanıcı Tarafı
- 🏠 **Ana Sayfa** — Slider, son bloglar, en popüler 10 blog ve en beğenilen yerler
- 📝 **Blog Listesi** — Tüm blog yazıları, son bloglar ve son yorumlar sidebar
- 📖 **Blog Detay** — Tam blog içeriği, yorumlar ve yorum yapma formu
- 💬 **Yorum Sistemi** — Blog yazılarına yorum ekleme
- 📞 **İletişim Formu** — Ad, email, konu ve mesaj alanları ile veritabanına kayıt
- ℹ️ **Hakkımızda** — Site sahibi hakkında bilgi

### 🗺️ Rehber Sayfaları
- 🏨 **Oteller** — Dünyanın en prestijli 6 oteli (Burj Al Arab, The Ritz, Peninsula vb.)
- 🍽️ **Restoranlar** — Dünyanın en iyi 6 restoranı (Noma, El Celler de Can Roca vb.)
- 🏛️ **Müzeler** — Dünyanın en önemli 6 müzesi (Louvre, British Museum vb.)
- ✈️ **Uçuşlar** — Dünyanın en iyi 6 havayolu şirketi (Emirates, Singapore Airlines vb.)
- 🗺️ **Aktiviteler** — Dünyadan 6 unutulmaz seyahat deneyimi
- 🚗 **Araç Kiralama** — Dünyanın en güvenilir 6 araç kiralama firması

### 🔐 Admin Paneli
- Blog ekleme, düzenleme, silme ve detay görüntüleme
- Yorum listeleme, güncelleme ve silme
- İletişim mesajlarını görüntüleme ve silme
- Hakkımızda içeriği ekleme, düzenleme ve silme
- Admin Sayfası `[Authorize]` ile korunmaktadır

## ⚙️ Kurulum

### Gereksinimler
- Visual Studio 2019 veya üzeri
- .NET Framework 4.x
- SQL Server 2019 veya üzeri
- SQL Server Management Studio (SSMS)

### Adımlar

**1. Projeyi klonlayın:**
```bash
git clone https://github.com/MelikCvk/TravelTripProje.git
```

**2. Veritabanını restore edin:**
- SSMS'i açın
- Databases → Restore Database
- `TravelData.bak` dosyasını seçin
- Restore işlemini tamamlayın

**3. Connection String'i güncelleyin:**

`Web.config` dosyasında aşağıdaki satırı kendi SQL Server bilgilerinize göre düzenleyin:

```xml
<connectionStrings>
  <add name="Context" 
       connectionString="Data Source=SUNUCU_ADI;Initial Catalog=TravelData;Integrated Security=True" 
       providerName="System.Data.SqlClient" />
</connectionStrings>
```

**4. Projeyi çalıştırın:**
- Visual Studio'da `F5` veya `Ctrl+F5`
